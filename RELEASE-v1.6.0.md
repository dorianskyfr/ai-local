## 🚀 AI Local v1.6.0 — RÉVOLUTION : Un seul modèle partagé par TOUS les utilisateurs !

**NOUVEAU CONCEPT MONDIAL** : Plus de modèles isolés, plus de fragmentation. Désormais, **tous les utilisateurs partagent et améliorent le même modèle global**. Chaque conversation, chaque correction, chaque apprentissage profite à toute la communauté. C'est la première IA locale où **l'intelligence collective est littérale**.

### 💥 CHANGEMENT MAJEUR : Système de Modèle Collaboratif Unique

- **Un seul modèle pour tous** : Finis les modèles séparés. Tous les utilisateurs utilisent et améliorent **le même modèle global**
- **Synchronisation automatique** : Le modèle se met à jour au démarrage et toutes les 5 minutes
- **Contributions transparentes** : Tes améliorations locales sont automatiquement fusionnées et partagées avec la communauté
- **100% offline après sync** : Une fois synchronisé, tout fonctionne hors ligne sans dépendance externe
- **Anonymat garanti** : Les contributions sont anonymes (ID basé sur l'installation, pas de données personnelles)

### 🔧 Améliorations Techniques Majeures

#### Nouveau Moteur de Langage (brain.js - Réécriture Complète)
- **Markov Order 3** : Passage de l'ordre 2 à l'ordre 3 pour des réponses plus cohérentes et contextuelles
- **Mémoire Hiérarchique** : 
  - Mémoire à court terme (50 derniers messages)
  - Mémoire à long terme (15 000 entrées, ×5 par rapport à v1.5)
- **Système de Synonymes** : 12+ groupes de synonymes pour une meilleure compréhension
- **Concepts Connexes** : Expansion automatique des concepts liés pour des réponses plus riches
- **Suivi de Contexte** : Le modèle comprend et retient le fil de la conversation
- **Raisonnement Simple** : Capacité à répondre à des questions logiques basiques

#### Fonctions Mathématiques Étendues
- **sin, cos, tan, sqrt, log, exp, pow** : Toutes les fonctions mathématiques de base
- **Calcul exact** : Précision améliorée pour les opérations mathématiques

### ⚡ Suppressions Importantes

- **Qwen/LLM Externe Supprimé** : Plus aucune dépendance à des modèles externes (node-llama-cpp)
- **llm-main.js Supprimé** : Le système LLM externe a été complètement retiré
- **Moins de dépendances** : Application plus légère et plus rapide

### 🛡 Vie Privée et Sécurité

- **Conversations privées protégées** : Les messages marqués comme "toi" ne sont JAMAIS partagés
- **Pas de tokens GitHub requis** : La synchronisation fonctionne sans authentification
- **Données locales prioritaires** : Tes améliorations locales sont conservées même après sync
- **Backward compatible** : Les anciens modèles locaux peuvent être migrés
