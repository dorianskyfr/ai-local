## 🎉 AI Local v0.1.0 — Première version

Application de bureau Windows avec une interface de chat façon ChatGPT / Claude,
embarquant un modèle d'IA 100 % local qui s'entraîne tout seul.

### ✨ Nouveautés
- **Interface de chat** moderne : thème sombre, barre latérale, bulles de messages,
  indicateur de frappe, bouton « Nouvelle conversation ».
- **Modèle d'IA local** (chaîne de Markov d'ordre 2 avec repli d'ordre 1) : il apprend
  le vocabulaire et le style de chaque message que tu lui envoies, entièrement hors ligne.
- **Mode auto-entraînement** : un bouton dans la barre latérale lance une boucle où le
  modèle consolide la conversation, génère ses propres phrases, les note et renforce
  les meilleures. Statistiques en direct : cycles, vocabulaire, transitions, confiance.
- **Persistance** : le modèle est sauvegardé automatiquement et retrouvé au lancement suivant.
- **Réinitialisation** du modèle possible en un clic.
