## 🌐 AI Local v0.4.0 — Multi-sources, mises à jour automatiques et modèle communautaire

Le modèle apprend désormais de plusieurs sources internet **en parallèle**, l'app se met
à jour toute seule, et le modèle devient **communautaire** : hébergé sur GitHub, partagé
entre toutes les installations, sans jamais inclure les conversations.

### ✨ Nouveautés
- **Apprentissage multi-sources en parallèle** : Wikipédia, Vikidia (encyclopédie junior),
  Wikinews, Wiktionnaire, Wikisource (livres) et les actualités (France Info, Le Monde)
  sont interrogés **en même temps** à chaque cycle — l'apprentissage est beaucoup plus
  rapide. Un menu « Sources » permet de choisir une source précise ou toutes à la fois.
- **YouTube** : colle un lien de vidéo dans le champ Sujet et le modèle apprend ses
  **sous-titres** (français ou anglais). Les vidéos sans sous-titres accessibles sont
  signalées honnêtement.
- **Mise à jour automatique** : à chaque démarrage, l'app vérifie les releases GitHub ;
  si une nouvelle version existe, elle propose de télécharger et lancer le nouvel
  installateur — ton modèle et tes conversations sont conservés.
- **Modèle communautaire sur GitHub** :
  - au démarrage, chaque installation télécharge le modèle partagé
    (`shared-model/model.json`) et le fusionne avec le sien — une nouvelle installation
    ne repart jamais de zéro, une désinstallation ne fait rien perdre ;
  - à chaque **grande avancée** d'entraînement (300 nouvelles phrases), le modèle
    fusionné est publié sur GitHub (jeton GitHub à configurer dans la barre latérale) ;
  - **les conversations ne sont jamais partagées** : seuls le cerveau (vocabulaire,
    transitions, souvenirs sourcés) et le génome d'images voyagent, et les souvenirs
    personnels (« tu m'avais dit… ») sont exclus de l'export.
- **Chatbot moins bête** 🙂 :
  - il ne répète plus ta question en guise de réponse (garde anti-écho) ;
  - « salut », « ça va ? », « merci », « qui es-tu ? »… ont de vraies réponses variées
    qui s'appuient sur ses statistiques réelles ;
  - une question sans souvenir correspondant reçoit une réponse honnête qui t'invite
    à l'entraîner sur le sujet.
