## 🌍 AI Local v0.6.0 — Tout le web, vitesse Ultimate, scènes générées repensées

Le modèle ne se limite plus à des sources prédéfinies : il peut chercher lui-même sur
tout le web. La génération d'images/vidéos compose désormais des scènes vraiment
différentes selon le sujet, et une vitesse Ultimate est disponible pour les PC costauds.

### ✨ Nouveautés
- **Recherche libre sur tout le web** : nouvelle source « 🔎 Recherche libre sur tout
  le web (DuckDuckGo) », interrogée en parallèle avec les autres. Le modèle n'est plus
  limité aux sources prédéfinies — il trouve une page pertinente et en apprend le texte.
- **N'importe quelle page en sujet** : colle n'importe quel lien (article de blog,
  site d'actualité, page quelconque) dans le champ Sujet et le modèle en extrait le
  texte lisible, avec apprentissage automatique des images qu'elle contient.
- **Vimeo et Dailymotion** : en plus de YouTube, colle un lien Vimeo ou Dailymotion
  pour que le modèle apprenne son titre et sa description publique. Honnêteté : un
  modèle de texte local ne peut pas « regarder » une vidéo ni « écouter » un son —
  seules les métadonnées publiques (texte) sont exploitables, comme pour YouTube.
- **Vitesse Ultimate** : un cinquième palier, jusqu'à un cycle toutes les 0,7 s, pour
  les PC puissants (non recommandé automatiquement — sollicite fortement les sources).
- **Génération d'images/vidéos repensée** : le générateur compose désormais une scène
  différente selon le sujet — un vrai volcan avec cratère lumineux, coulées de lave et
  braises qui montent ; un océan avec vagues et écume ; une forêt avec silhouettes
  d'arbres et lucioles ; un ciel étoilé avec planète, anneaux, nébuleuses et comète.

### 🔒 Sécurité : pas de jeton unique intégré à l'app
Il a été demandé qu'un jeton GitHub unique, partagé par tout le monde, soit intégré
par défaut dans l'application pour rendre la contribution obligatoire et automatique.
**Ce n'est pas fait, volontairement** : un secret capable d'écrire sur GitHub, présent
dans un exécutable public, serait extractible par n'importe qui en quelques minutes
(l'app n'est ni compilée en code natif ni chiffrée — un `.exe` Electron est un simple
zip de fichiers JS/HTML lisibles). GitHub détecte et révoque automatiquement les jetons
qui fuitent publiquement, ce qui casserait le partage pour tout le monde du jour au
lendemain. Le téléchargement et la fusion du modèle communautaire restent, eux,
automatiques pour tout le monde sans exception (aucun changement, déjà le cas depuis
la v0.4). Pour **publier** ses propres avancées, chacun utilise son propre jeton
gratuit (2 minutes sur github.com/settings/tokens), verrouillé une fois enregistré.
