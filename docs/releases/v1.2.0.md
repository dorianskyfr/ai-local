## 🚀 AI Local v1.2.0 — Un vrai LLM local en option, le meilleur du 100 % local

La plus grosse évolution depuis la création de l'app : elle peut désormais faire
tourner un **véritable modèle de langage** (LLM) **entièrement sur ton PC**, via
llama.cpp — la référence mondiale de l'IA locale. Aucune donnée n'est envoyée
dans le cloud : le modèle est téléchargé une fois, puis tout fonctionne hors ligne.

### 🧠 Le cerveau LLM local (optionnel)
- **Vraies réponses générées** : conversation naturelle en français, questions
  ouvertes, explications, reformulations — plus seulement des faits cités.
- **Nourri par ta mémoire locale (RAG)** : les faits appris pendant tes
  entraînements sont automatiquement fournis au LLM, qui s'appuie dessus en
  priorité et cite ses sources. Ton modèle auto-appris n'est pas remplacé —
  il devient la base de connaissances du LLM.
- **Choix du modèle** dans la barre latérale : Qwen 2.5 (licence Apache 2.0)
  en 0,5 / 1,5 / 7 milliards de paramètres selon la puissance du PC — ou
  l'URL de n'importe quel fichier `.gguf` pour les utilisateurs avancés.
- **Téléchargement intégré** avec barre de progression et annulation ; le
  dernier modèle utilisé est rechargé automatiquement au démarrage.
- **Streaming en direct** : la réponse s'écrit mot à mot pendant la génération.
- **Sans LLM, rien ne change** : l'app fonctionne exactement comme en v1.1
  (moteur de faits, recherche instantanée, aveu honnête). Le LLM est un étage
  au-dessus, jamais une dépendance.

### 🔩 Sous le capot
- Nouveau module d'inférence dans le processus principal (llama.cpp via
  node-llama-cpp, CPU, aucune dépendance GPU) avec gestion propre du cycle de
  vie : téléchargement, chargement, session par conversation, arrêt en cours
  de génération, déchargement.
- Les outils exacts (calculatrice, date/heure) gardent la priorité sur le
  LLM : une question d'arithmétique reçoit toujours la réponse déterministe.
- Si le sujet est inconnu de la mémoire, la recherche web instantanée
  alimente d'abord la mémoire, puis le LLM répond avec ces faits.
- En cas d'erreur du LLM (mémoire insuffisante, fichier corrompu…), repli
  automatique et silencieux sur le moteur de faits.

### 🔒 Toujours local, toujours honnête
Les modèles proposés sont des modèles ouverts redistribuables (Apache 2.0),
téléchargés depuis leur dépôt officiel. La consigne système du LLM impose de
citer les faits fournis et d'admettre ce qu'il ne sait pas. Tes conversations,
ta mémoire personnelle et le LLM lui-même ne quittent jamais ta machine.
