## 🏆 AI Local v1.5.0 — La version finale : audit complet, zéro bug connu

Version de consolidation et de finition : tout le code a été relu fichier par
fichier, chaque bug trouvé a été corrigé, et l'app entière est couverte par une
grande suite de tests de bout en bout (chat, calcul, mémoire, conversations,
images, galerie, entraînement, vitesse personnalisée, panneau LLM, persistance
après redémarrage) — 57 vérifications automatisées, toutes au vert.

### 🛠 Corrections (issues de l'audit complet)
- **Envois concurrents bloqués** : appuyer sur Entrée pendant qu'une réponse
  était en cours déclenchait un second envoi en parallèle (double indicateur
  de frappe, générations en collision). Un garde anti-réentrance l'empêche.
- **Plus aucune fuite de contexte entre conversations** : le sujet de suivi du
  rappel de faits et l'historique du LLM sont remis à zéro en changeant de
  conversation, en en créant une nouvelle ou en supprimant celle affichée.
- **Questions d'attribut sans sujet assainies** : une question composée
  uniquement de mots d'attribut sans sujet identifiable et sans contexte
  hérité ne cite plus jamais un souvenir par coïncidence — réponse honnête.
- **Nouvelle conversation pendant une génération LLM** : le changement de
  session est maintenant mémorisé et appliqué dès la fin de la génération au
  lieu d'être perdu (l'historique de l'ancienne conversation pouvait fuir).
- **Numéro de version enfin unique** : le badge de la barre latérale, le titre
  de la fenêtre et la présence Discord affichaient des versions figées datant
  de versions précédentes. Tout vient désormais d'une source unique (la
  version réelle de l'app) — plus jamais de numéro périmé.
- **Téléchargement du grand modèle LLM fiabilisé** : le dépôt officiel scinde
  les gros fichiers en plusieurs morceaux, ce qui aurait fait échouer le
  téléchargement intégré — remplacé par un miroir mono-fichier éprouvé
  (même modèle, même licence Apache 2.0).

### ✅ Vérifié de bout en bout
- 24 tests du moteur de faits (toutes les régressions des versions passées
  rejouées), 11 tests du module LLM (catalogue, IPC, tous les cas d'erreur),
  22 vérifications de l'app réelle dans un navigateur automatisé — y compris
  la persistance du modèle après un redémarrage complet.
- Aucune erreur JavaScript sur l'ensemble du parcours utilisateur.

### 🐧 Dernier cadeau : la version Linux
L'app est désormais aussi disponible pour Linux, dans la même release :
l'**AppImage** fonctionne sur toutes les distributions sans installation
(`chmod +x` puis double-clic), le **.deb** s'installe sur Ubuntu/Debian.
Mêmes fonctionnalités qu'en version Windows, LLM local compris.

### 📦 Fichiers
| Fichier | Description |
|---|---|
| `AI-Local-1.5.0-portable.exe` | Windows, version portable — à lancer directement, aucune installation |
| `AI Local Setup 1.5.0.exe` | Windows, installateur classique (recommandé pour les mises à jour auto) |
| `AI-Local-1.5.0.AppImage` | Linux, portable toutes distributions — `chmod +x` puis lancer |
| `ai-local_1.5.0_amd64.deb` | Linux, paquet Ubuntu/Debian — `sudo apt install ./ai-local_1.5.0_amd64.deb` |

> 💡 Si tu as une version précédente installée (installateur), elle te proposera
> cette mise à jour automatiquement au prochain démarrage.
